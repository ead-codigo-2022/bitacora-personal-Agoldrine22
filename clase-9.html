Revisión Proyectos Grupales

Correciones generales

Corregir vínculos de archivos internos:

  • Estilos CSS que no se cargan
  • Archivos HTML mal vinculados
  • Links dentro de una misma página que no funcionan

Seguir recomendaciones de nombres de carpetas y archivos

  • Minúsculas, palabras separadas por guión.

Avanzar con contenidos

  • Notas de clases
  • Contar de cada uno
  • No debe haber contenido de maqueta

Cuidar aspectos de diseño
